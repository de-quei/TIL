## 인터넷이란?
* 정의
  * Inter + net (내부 + 망)
  * 컴퓨터들이 연결된 통신망을 일컫는다.
  * TCP/IP를 이용하여 컴퓨터들을 연결하는데, 이 연결들이 모여  거대한 네트워크를 만드는 것 혹은 네트워크 안 네트워크를 형성하는 모든 일련의 과정
  
  </br>

  ## TCP/IP
  > 지역네트워크(LAN), 광역네트워크(WAN)에서 원활한 통신을 가능하도록 하기위한 통신규약.
  
* 하드웨어, 운영체제, 접속매체에 관계없이 동작할 수 있다는 개방성을 특징으로 가지고 이러한 **개방성**이 인터넷 통신에 사용된다.

1. TCP ( Transmission  Control Protocol (전송 제어 프로토콜) )
   *  클라이언트와 서버간에 데이터를 신뢰성있게 전달하기 위해 만들어진 프로토콜
   *  LAN, WAN, 인트라넷, 인터넷 등 컴퓨터에서 실행되는 프로그램 간에 일련의 데이터를 안정적으로 순서대로 에러없이 데이터를 교환하게 한다.
  
</br>

2. IP ( Internet Protocol ( 인터넷 프로토콜 ))
   * 네트워크 상에는 컴퓨터 고유 주소가 존재하는데, 컴퓨터의 주소는 인터넷에 접속할 때 컴퓨터 각각에 부여받는다. (4바이트)
   
   * ip 주소 얻는 법 ! ( cmd )
      1. windows
        ```
        ipconfig
        ```
      2. unix 
        ```
        ifconfig
        ```

**Result** ==> 인터넷이란 각 컴퓨터들간의 TCP/IP 통신 프로토콜을 이용해서 서로 데이터를 주고 받도록한 네트워크를 말함.
